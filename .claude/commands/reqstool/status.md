Show reqstool requirements traceability status.

**Argument**: Optional module — `system`, `core`, `app`, or `all` (default).

Run `reqstool status local -p <path>` for the selected module(s) and summarize results.

| Module | Path |
|--------|------|
| system | `docs/reqstool` |
| core   | `core/docs/reqstool` |
| app    | `app/docs/reqstool` |
