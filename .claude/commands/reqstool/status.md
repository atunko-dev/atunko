description: Show requirements traceability status for one or all modules
category: Requirements
tags: [reqstool, requirements, traceability, status]

Show reqstool requirements traceability status.

**Argument**: Optional module — `system`, a module name from `.reqstool-ai.yaml`, or `all` (default).

Read `.reqstool-ai.yaml` to determine available modules and their paths, then run `reqstool status local -p <path>` for the selected module(s) and summarize results.
