## ADDED Requirements

### Requirement: WEB_0001.17
The system SHALL implement WEB_0001.17.

#### Scenario: SVC_WEB_0001.28
- **WHEN** the Web UI is loaded
- **THEN** the system SHALL pass SVC_WEB_0001.28.

#### Scenario: SVC_WEB_0001.29
- **WHEN** recipes are selected and the user opens the export dialog
- **THEN** the system SHALL pass SVC_WEB_0001.29.

#### Scenario: SVC_WEB_0001.30
- **WHEN** no recipes are selected and the user opens the export dialog
- **THEN** the system SHALL pass SVC_WEB_0001.30.

#### Scenario: SVC_WEB_0001.31
- **WHEN** the user switches the format in the export dialog
- **THEN** the system SHALL pass SVC_WEB_0001.31.
