= atunko
:description: An open-source TUI + CLI tool for OpenRewrite

**atunkọ** (Yoruba) — _the act of rewriting/rebuilding_, from *atun* (again) + *kọ* (to write/build)

An open-source TUI + CLI tool providing a rich, interactive developer experience for
https://docs.openrewrite.org[OpenRewrite] — recipe discovery, browsing, configuration,
execution, and saveable run configurations — all from the terminal.

NOTE: "OpenRewrite" is a registered trademark of Moderne, Inc. This project is not affiliated
with or endorsed by Moderne.

== Features

* **Recipe Discovery** — Browse and search all available OpenRewrite recipes
* **Recipe Execution** — Run recipes against your project from the command line
* **Run Configurations** — Save and load recipe setups as portable `.atunko.yml` files
* **Project Scanning** — Gradle and Maven project support for classpath resolution

== Quick Start

[source,bash]
----
# Requires Java 25+
./gradlew build

# Launch TUI
./gradlew :app:run

# CLI commands
./gradlew :app:run --args="discover"
./gradlew :app:run --args="discover --search 'spring boot'"
./gradlew :app:run --args="run -r org.openrewrite.java.RemoveUnusedImports --project-dir ."
----

See xref:cli.adoc[CLI Usage] for full command reference.
